# FRAC!
Descripcion: El objetivo de nuestro proyecto esta basado en la promocion 
de la educacion de fracciones de manera simple, dinamica y divertida,
que consta de teoria y juegos diseñado para los niños

## Autores
El sitio fue diseñado como un proyecto para la materia Laboratorio de Computación 2 (Universidad Católica de Córdoba)
por Dominguez Mendoza Rocio y Nanfara Nara Abril

## Tecnologia
el proyecto consta de:
* HTML: dividido en secciones (Home, teoría, ejemplos y práctica) bajo la estructura de filas que envuelven columnas, contiene una barra de navegación y unn footer.
* CSS: define los estilos empleados en el html, miEstilo.css fue personalizado exclusivamente para la página, mientras que Style.css
se utilizo para la base (diseñado para el template).
* Images: Carpeta que contiene las imagenes empleadas.
* JavaScrip: será desarrollado con profundidad posteriormente.
